# 🐶 We Care

Este é um projeto de uma landing page desenvolvido durante os estudos no DevClub, com o objetivo de praticar HTML5 e CSS3, criando uma interface moderna e responsiva para um serviço fictício de cuidados com pets.

O projeto foi desenvolvido utilizando apenas HTML5 e CSS3, com foco na organização do código, estilização e boas práticas de desenvolvimento Front-End.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## 📄 Funcionalidades

- Landing page moderna e intuitiva.
- Estrutura semântica em HTML.
- Estilização utilizando CSS.
- Layout organizado e de fácil navegação.

## 🎯 Objetivo

Praticar conceitos fundamentais de desenvolvimento Front-End, como:

- Estruturação de páginas com HTML.
- Estilização com CSS.
- Posicionamento de elementos.
- Organização e legibilidade do código.

## 📂 Como executar

1. Clone este repositório:

```bash
git clone https://github.com/seu-usuario/we-care.git
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` no navegador.
